Neon Stick Fighter

Neon Stick Fighter is a fluid side-scrolling martial arts brawler set in a neon-lit cyberpunk city. Battle wave after wave of incoming stickman fighters using punches, kicks, and powerful combo chains. Each enemy staggers, stumbles, and reacts with ragdoll physics to your attacks. Progress through increasingly dangerous fighter types including armored elites and rapid dual-weapon attackers. How many waves can you survive?

🎮 How to Play

Face enemies approaching from the right. Attack with punch and kick buttons to build combos. Block incoming strikes to preserve your health bar. Defeat all enemies in each wave to progress.

  • Desktop — Move: A / D or Left / Right Arrow Keys
  • Desktop — Punch: J key  |  Kick: K key  |  Block: L key
  • Mobile: On-screen Punch, Kick, and Block buttons
  • Combo: Chain punch → punch → kick for a power finisher

⚡ Game Features

  • Fluid punch-kick-combo system with ragdoll physics
  • Multiple enemy classes: brawlers, rushers, armored elites
  • Glowing neon particle impact effects on every hit
  • Progressive wave difficulty with increasing enemy count

🏆 Pro Tips

  • Block just before an enemy's attack lands for a perfect parry bonus
  • Use kick attacks to knock enemies back and create spacing
  • Focus on one enemy at a time; fighting multiple simultaneously is risky
  • Chain punch → punch → kick for maximum damage per combo
