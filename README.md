# Arcade Hub
A classic game arcade for users built entirely in the browser. 

## Core Development Rules
1. **Always Mobile Responsive:** Every UI element, game canvas layout, menu modal, and navigation button MUST be tested and designed to be flawlessly responsive on narrow mobile touch-screens. Elements cannot break styling, overlap incorrectly, or wrap poorly.
2. **Device Input Compatibility:** All games must naturally switch between and support both keyboard Desktop controls and Mobile swipe/touch controls.
