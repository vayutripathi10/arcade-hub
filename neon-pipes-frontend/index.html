Neon Pipes

Neon Pipes is a cerebral, fast-paced logic puzzle game set in a glowing cyberpunk grid. Rotate disconnected pipe segments to forge a complete circuit flowing neon plasma from the input source node all the way to the output destination node. Every level increases complexity with new pipe shapes, tighter grids, and a relentless countdown timer. The ultimate test of spatial reasoning and quick thinking.

🎮 How to Play

Click or tap any pipe tile to rotate it 90 degrees clockwise. Connect every pipe into a single unbroken flow from the green source to the red destination before the timer runs out.

  • Desktop: Left-click any pipe tile to rotate it 90° clockwise
  • Mobile: Tap any tile to rotate it
  • Objective: Connect source (green) to destination (red) using all pipe segments
  • Timer: Each level has a countdown — finish before time expires

⚡ Game Features

  • Multiple grid sizes scaling in difficulty
  • Straight, curved, and T-junction pipe types
  • Ticking countdown timer per level
  • Neon glow visual feedback on connected pipes

🏆 Pro Tips

  • Work backwards from the destination node to trace the correct route
  • Lock in corner pipes first — they have the fewest rotation options
  • Watch the clock: prioritize the main route before filling side branches
  • Straight pipes in the center are usually the trickiest — do them last
